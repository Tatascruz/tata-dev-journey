# Algoritmo-livro

Repositório para armazenar todos os exercícios feitos durante a leitura do livro **"Entendendo Algoritmos"**.

Aqui, cada capítulo do livro terá:
- código em Go
- explicações
- simulações
- anotações importantes

## Estrutura
- capitulo1 -> Pesquisa binária 
- capitulo2 -> Ordenação por seleção (em breve)
- capitulo3 -> Recursão (em breve)

## Como executar os códigos

go run nome-do-arquivo.go
